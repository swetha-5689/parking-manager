SmartPark Elevator Terminal Setup:

In order to run the Elevator Terminal code, Node.js and npm must be installed.
If you wish to run the SmartPark Elevator Terminal without downloading these
dependencies, please visit https://smartpark-elevator-terminal.herokuapp.com/

With the dependencies installed, open up your command prompt and
navigate to the location where the SmartPark Elevator Terminal folder was downloaded. 
This folder is known as "parking-garage-elevator-terminal". When this folder is reached,
use "npm start" in order to start the application. The application will take a few seconds
to start and will then open up in your default browser.
